MeshStandardMaterial en Three.js

⬅ Volver al menú

Ejemplo de código

// Crear una geometría (por ejemplo, un toroide)
var geometry = new THREE.TorusKnotGeometry(2, 0.6, 100, 16);

// Crear MeshStandardMaterial
// Es el material más moderno y realista (PBR). Requiere luces.
var material = new THREE.MeshStandardMaterial({
    color: 0xffff00,    // Color base (amarillo)
    metalness: 0.5,     // Qué tan metálico es (0.0 = no metálico, 1.0 = metal puro)
    roughness: 0.2      // Qué tan rugoso es (0.0 = liso espejo, 1.0 = mate total)
});

// Crear el objeto
var objeto = new THREE.Mesh(geometry, material);
scene.add(objeto);

Descripción

El MeshStandardMaterial es el material estándar de la industria hoy en día. Utiliza un modelo de Renderizado Basado en Física (PBR) para lograr resultados muy realistas.

  • Sus dos propiedades principales para controlar su apariencia son metalness (metalicidad) y roughness (rugosidad).
  • Con roughness: 0, la superficie es como un espejo; con roughness: 1, es completamente mate.
  • Es más costoso computacionalmente que el Phong o el Basic, pero ofrece una calidad visual superior y se comporta de manera más predecible bajo diferentes condiciones de luz.
