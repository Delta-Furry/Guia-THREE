Cono en Three.js

⬅ Volver al menú

Ejemplo de código

// Crear geometría del cono
var geometry = new THREE.ConeGeometry(5, 10, 32);

// Crear material básico
var material = new THREE.MeshBasicMaterial({ color: 0xff9900 });

// Crear el cono y añadirlo a la escena
var cono = new THREE.Mesh(geometry, material);
scene.add(cono);

Descripción

Para crear un cono en Three.js usamos ConeGeometry: ConeGeometry(radio, altura, segmentosRadiales).

  • Radio: 5 → tamaño de la base del cono.
  • Altura: 10 → qué tan alto es el cono.
  • Segmentos radiales: 32 → cortes alrededor del círculo de la base.

Con más segmentos, la base del cono se ve más circular; con menos, más poligonal.
