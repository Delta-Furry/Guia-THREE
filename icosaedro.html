Icosaedro en Three.js

⬅ Volver al menú

Ejemplo de código

// Crear geometría del Icosaedro (Dado de 20 caras)
// IcosahedronGeometry(radio, detalle)
var geometry = new THREE.IcosahedronGeometry(4, 0);

// Crear material básico (Color Morado)
var material = new THREE.MeshBasicMaterial({ color: 0xaa00ff, wireframe: false });

// Crear el icosaedro y añadirlo a la escena
var icosaedro = new THREE.Mesh(geometry, material);
scene.add(icosaedro);

Video demostrativo

Descripción

Para crear esta figura de 20 caras triangulares usamos IcosahedronGeometry: IcosahedronGeometry(radio, detalle).

  • Radio: 4 (El tamaño general).
  • Detalle: 0 (Si dejas esto en 0, obtienes el icosaedro clásico. Si pones 1 o más, Three.js añade más triángulos y empieza a parecerse a una esfera).
