Icosaedro en Three.js

⬅ Volver al menú

Ejemplo de código

// Crear geometría del icosaedro
var geometry = new THREE.IcosahedronGeometry(5);

// Crear material básico
var material = new THREE.MeshBasicMaterial({ color: 0x33ffcc });

// Crear el icosaedro y añadirlo a la escena
var icosaedro = new THREE.Mesh(geometry, material);
scene.add(icosaedro);

Video demostrativo

Descripción

Para crear un icosaedro en Three.js usamos IcosahedronGeometry: IcosahedronGeometry(radio).

  • Radio: 5 → tamaño del icosaedro desde el centro hasta los vértices.

El icosaedro es un poliedro con 20 caras triangulares, todas iguales y simétricas.
