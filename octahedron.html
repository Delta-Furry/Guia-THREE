Octaedro en Three.js

⬅ Volver al menú

Ejemplo de código

// Crear geometría del octaedro
var geometry = new THREE.OctahedronGeometry(5);

// Crear material básico
var material = new THREE.MeshBasicMaterial({ color: 0x00ccff });

// Crear el octaedro y añadirlo a la escena
var octaedro = new THREE.Mesh(geometry, material);
scene.add(octaedro);

Video demostrativo

Descripción

Para crear un octaedro en Three.js usamos OctahedronGeometry: OctahedronGeometry(radio).

  • Radio: 5 → tamaño del octaedro desde el centro hasta los vértices.

El octaedro es un poliedro con 8 caras triangulares iguales y simétricas.
