Ejemplo de código

// Crear una geometría (por ejemplo, una esfera)
var geometry = new THREE.SphereGeometry(3, 32, 32);

// Crear MeshPhongMaterial
// ¡IMPORTANTE! Este material NECESITA luces en la escena para verse bien.
var material = new THREE.MeshPhongMaterial({
    color: 0x0000ff,    // Color base (azul)
    specular: 0xffffff, // Color del brillo (blanco)
    shininess: 100      // Intensidad del brillo (más alto = más pequeño y nítido)
});

// Crear el objeto
var esfera = new THREE.Mesh(geometry, material);
scene.add(esfera);

Descripción

El MeshPhongMaterial es un material que simula superficies brillantes y pulidas. A diferencia del material básico, este sí reacciona a la luz.

  • Calcula la luz en cada píxel, lo que le da una apariencia suave.
  • Su característica principal es el brillo especular (el punto de luz brillante que ves en superficies como el plástico o la cerámica pulida).
  • Puedes controlar el color y la intensidad de ese brillo con las propiedades specular y shininess.
