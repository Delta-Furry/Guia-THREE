Pirámide en Three.js

⬅ Volver al menú

Ejemplo de código

// ¡Truco! Usamos ConeGeometry con solo 4 lados
// ConeGeometry(radio, altura, segmentosRadiales)
var geometry = new THREE.ConeGeometry(4, 5, 4);

// Crear material básico (Color Naranja)
var material = new THREE.MeshBasicMaterial({ color: 0xff7700, wireframe: false });

// Crear la pirámide y añadirla a la escena
var piramide = new THREE.Mesh(geometry, material);
scene.add(piramide);

Descripción

Three.js no tiene una geometría llamada "Pyramid". Para crear una, usamos ConeGeometry y definimos que solo tenga 4 segmentos radiales.

  • Radio: 4 (Tamaño de la base).
  • Altura: 5 (Altura de la punta).
  • Segmentos Radiales: 4 (Esto crea la base cuadrada).

Si cambias el 4 por un 3, ¡obtendrías un tetraedro!
