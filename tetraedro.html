Tetraedro en Three.js

⬅ Volver al menú

Ejemplo de código

// Crear geometría del Tetraedro (Pirámide triangular)
// TetrahedronGeometry(radio, detalle)
var geometry = new THREE.TetrahedronGeometry(4, 0);

// Crear material básico (Color Magenta)
var material = new THREE.MeshBasicMaterial({ color: 0xff0066, wireframe: false });

// Crear el tetraedro y añadirlo a la escena
var tetraedro = new THREE.Mesh(geometry, material);
scene.add(tetraedro);

Descripción

El Tetraedro es la figura 3D más simple, compuesta por 4 caras triangulares. Usamos: TetrahedronGeometry(radio, detalle).

  • Radio: 4 (El tamaño desde el centro a los vértices).
  • Detalle: 0 (El valor 0 crea la forma básica de 4 caras. Si aumentas este número, se subdivide y se vuelve más esférico).
