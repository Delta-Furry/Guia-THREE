Ejemplo básico de un Cuadrado

En Three.js no existe un objeto llamado Cuadrado como tal, pero podemos representarlo con un plano (PlaneGeometry) con medidas iguales de ancho y alto.


// Crear escena, cámara y renderizador
const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(75, window.innerWidth/window.innerHeight, 0.1, 1000);
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

// Geometría del cuadrado (ancho, alto)
const geometry = new THREE.PlaneGeometry(5, 5); // cuadrado de 5x5

// Material básico
const material = new THREE.MeshBasicMaterial({ color: 0x00ff00, side: THREE.DoubleSide });

// Crear malla
const square = new THREE.Mesh(geometry, material);
scene.add(square);

// Cámara y animación
camera.position.z = 10;

function animate() {
  requestAnimationFrame(animate);
  square.rotation.y += 0.01; // giro en Y
  square.rotation.x += 0.01; // giro en X
  renderer.render(scene, camera);
}
animate();

Descripción

En PlaneGeometry(ancho, alto), los primeros dos parámetros son:

  • Ancho: define la medida horizontal.
  • Alto: define la medida vertical.

En este ejemplo usamos un cuadrado de 5x5. Si quisieras un rectángulo, podrías poner diferentes valores, por ejemplo PlaneGeometry(8, 4).
