Cuadrado en Three.js

⬅ Volver al menú

Ejemplo de código

// Crear geometría del cuadrado
var geometry = new THREE.PlaneGeometry(5, 5);

// Crear material básico
var material = new THREE.MeshBasicMaterial({ color: 0x00ff00, side: THREE.DoubleSide });

// Crear el cuadrado y añadirlo a la escena
var cuadrado = new THREE.Mesh(geometry, material);
scene.add(cuadrado);

Video demostrativo

Descripción

Para crear un cuadrado en Three.js usamos PlaneGeometry(ancho, alto).
En este caso es (5, 5), lo que significa:

  • Ancho: 5
  • Alto: 5

Así obtenemos un cuadrado de 5x5 unidades en el espacio 3D.
