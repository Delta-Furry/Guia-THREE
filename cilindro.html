Cilindro en Three.js

⬅ Volver al menú

Ejemplo de código

// Crear geometría del cilindro
var geometry = new THREE.CylinderGeometry(3, 3, 8, 32);

// Crear material básico
var material = new THREE.MeshBasicMaterial({ color: 0x00ffff });

// Crear el cilindro y añadirlo a la escena
var cilindro = new THREE.Mesh(geometry, material);
scene.add(cilindro);

Descripción

Para crear un cilindro en Three.js usamos CylinderGeometry: CylinderGeometry(radioSuperior, radioInferior, altura, segmentosRadiales).

  • Radio superior: 3
  • Radio inferior: 3
  • Altura: 8
  • Segmentos radiales: 32

Si el radio superior y el inferior son diferentes, se forma un tronco de cono.
