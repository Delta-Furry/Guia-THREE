Ejemplo de código

// Crear un cubo de prueba
var geometry = new THREE.BoxGeometry(5, 5, 5);

// Crear material Phong (responde a luces con reflejos brillantes)
var material = new THREE.MeshPhongMaterial({ color: 0x0000ff, shininess: 100 });

// Crear el cubo y añadirlo a la escena
var cube = new THREE.Mesh(geometry, material);
scene.add(cube);

// Se necesita luz para que se vean los reflejos
var light = new THREE.PointLight(0xffffff, 1);
light.position.set(10, 10, 10);
scene.add(light);

Descripción

El MeshPhongMaterial es un material que responde a la luz con reflejos brillantes, simulando superficies pulidas o metálicas.

  • color: color base del objeto.
  • shininess: controla la intensidad del brillo (ejemplo: 100 = muy brillante).
  • Requiere luces en la escena para mostrar sus reflejos.

Ideal para objetos que deben lucir con brillo, como metales, plásticos o superficies pulidas.
