Toroide (Dona) en Three.js

⬅ Volver al menú

Ejemplo de código

// Crear geometría del Toroide (Dona)
// TorusGeometry(radio, tubo, segmentosRadiales, segmentosTubulares)
var geometry = new THREE.TorusGeometry(3, 1, 16, 48);

// Crear material básico (Color Turquesa)
var material = new THREE.MeshBasicMaterial({ color: 0x00aabb, wireframe: false });

// Crear el toroide y añadirlo a la escena
var toroide = new THREE.Mesh(geometry, material);
scene.add(toroide);

Descripción

Para crear una dona o toroide usamos TorusGeometry: TorusGeometry(radio, tubo, segRadiales, segTubulares).

  • Radio: 3 (El tamaño general del anillo).
  • Tubo: 1 (El grosor de la dona).
  • Segmentos Radiales: 16 (Qué tan redondo es el tubo).
  • Segmentos Tubulares: 48 (Qué tan redondo es el anillo).
